* Normal模式下，如何删除单个字符、单个单词、从当前光标位置一直删除到行尾、单行、当前行开始向下数N行？

    删除单个字符：`x`

    删除单个单词：`dw` 或者 `d1w`

    从当前光标位置一直删除到行尾：`d$`

    删除单行：`dd`

    删除当前行开始向下数N行：`Ndd`
